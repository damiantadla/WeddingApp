## Run Locally

Clone the project

```bash
  git clone https://github.com/damiantadla/WeddingApp.git
```

Go to the project directory

```bash
  cd WeddingApp
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
